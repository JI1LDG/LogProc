このソフトウェアはCTESTWINやZLOGで作成したログファイルを元に、提出用の電子ログの作成を支援するものです。
質問等ありましたら、山口(JI1LDG, owner@ji1ldg.xyz)まで。
出力されたデータの信頼性は保証しかねます。

無線局情報の検索に総務省の「無線局等情報検索(http://www.tele.soumu.go.jp/musen/SearchServlet?pageID=1)」を使用しています。

※Ver0.8.51より設定ファイルに変更があります。古い設定ファイルを読み込むと変換されます。
一部項目が読み込めなくなっているので、手動で入力をお願いします。


※ALL JA, 東京コンテスト, オール神奈川, 山梨コンテスト, 6m and Down, フィールドデー, 電信電話記念日コンテスト, ACAG, 関東UHF対応。
手持ちのログ(電話部門・マルチオペ・オールバンド)を使って確認をしておりますが、不具合が起こる可能性もあります。
※*.area.xmlは消したり編集したりしないようにしてください。
エラーチェック等はあまりしてません。詳しい状況（使用ファイル、操作）を伝えて頂けると助かります。
ログファイルのフォーマットさえ分かれば、他の形式にも対応できるかもしれません。

最低限必要な機能を入れたはずです。
不具合が残っていたり、突然落ちることがあるかもしれません。
処理時間をなるべく短くするために、RadioStaion.dbには検索した無線局の情報を格納しています。無線局等情報検索で簡単に調べられるものですが、取り扱いには注意してください。


・動作環境
Windows7以降
Windows7で使用する場合は、最低でも.NET Framework 4.5, 可能な限り.NET Framework 4.6のインストールを行ってください。
Windows8以降は標準で.NET Framework 4.5インストールされているかもしれません。
インターネットへの接続が必要です。
共有フォルダなどネットワーク経由では使用できません。必ずローカルに落としてから使用してください。
Windows7 professional 64-bit環境で動作確認をしています。
